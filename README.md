# Plan Tracker 📅

A simple, beautifully designed, and strict plan tracking system to monitor what you plan versus what you successfully complete. Built with modern web aesthetics featuring glassmorphism and smooth interactions.

## 🌟 Features

* **Minimalist & Focused**: Direct interface to log time slots and tasks without unnecessary bloat.
* **Modern UI/UX**: Premium aesthetic with dark glassmorphic components, responsive gradients, and modern typography.
* **Local Storage Integration**: All your tasks are saved locally in your browser. No server or login required!
* **Auto-Day Calculation**: Enter a date and the application automatically figures out the day of the week.
* **One-Click Status Toggling**: Easily mark tasks as \`Done\` or \`Not Done\` with visual color indicators.
* **Categorized Planning**: Keep track of daily time slots alongside Weekly Plans, Monthly Plans, and Special/Urgent Tasks.

## 🚀 Technologies Used

This project relies purely on front-end web foundations for maximum performance and portability:

* **HTML5**: Semantic structure.
* **CSS3**: Variables, Flexbox/Grid, Glassmorphism (`backdrop-filter`), and CSS animations.
* **Vanilla JavaScript (ES6)**: DOM manipulation and `localStorage` state management without any heavy frameworks.

## 🛠️ Installation & Usage

Since this is a static web application, you don't need to install any dependencies or run a build step.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yo-ezmir/schedule_Tracker.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd schedule_Tracker
   ```
3. **Open the app:**
   Simply double-click the `index.html` file to open it in your default web browser, or serve it using any local development server (like VS Code Live Server).

## 💡 How It Works

1. Open the application.
2. Fill out the "Add Block" form with the Date, Time Slot, and your Planned Task. 
3. (Optional) Provide context with Weekly/Monthly plans or mark special urgent tasks.
4. Click **Add Plan**.
5. Your plan will instantly appear in the table below. Click the status button to toggle it between \`Not Done\` (Red) and \`Done\` (Green) as you complete your goals.
